Tema premium para desarrollo territorial regenerativo con diseño moderno y bloques Gutenberg editables.

== Description ==

Sueño Andino es un tema WordPress premium diseñado específicamente para organizaciones de desarrollo territorial regenerativo. Incluye todas las secciones necesarias para una landing page corporativa moderna y profesional.

= Características principales =

* Diseño premium y moderno
* Completamente responsive
* Bloques Gutenberg editables
* Optimizado para SEO
* Accesibilidad AA+
* Formularios de contacto integrados
* Sistema de blog completo
* Colores personalizados
* Tipografía Inter optimizada

= Secciones incluidas =

* Hero con estadísticas
* Timeline de la empresa
* Sección "¿Por qué Sueño Andino?"
* Servicios con CTA
* Testimonios e impacto
* Equipo y directorio
* Formulario de contacto
* Header y Footer completos

= Instalación =

1. Subir el tema a wp-content/themes/sueno-andino/
2. Activar el tema en Apariencia > Temas
3. Crear páginas usando los bloques Gutenberg incluidos
4. Personalizar contenido según necesidades
