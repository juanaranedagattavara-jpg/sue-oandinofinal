Estándar de Calidad — Sueño Andino

Checklist operativo para asegurar consistencia, rendimiento y marca.

UX / UI

  • Tipografía y paleta desde theme.json
  • Contraste AA mínimo 4.5:1
  • Jerarquía clara: kicker → H1/H2 → body → CTA
  • Espaciados fluidos: clases .section y .container
  • Responsive design mobile-first
  • Estados de hover y focus visibles

Contenido

  • Copy premium, tono institucional inspirador
  • CTAs funcionales a Calendly/WhatsApp/PDF
  • Logos optimizados y sin pixelación
  • Imágenes de alta calidad y relevantes
  • Textos sin errores ortográficos

Rendimiento

  • LCP < 2.5s (desktop) / < 3s (mobile)
  • Imágenes en WebP/AVIF
  • Lazy load para videos/imágenes
  • CSS y JS minificados
  • CDN para assets estáticos

SEO

  • Títulos y metas personalizadas
  • Open Graph y JSON-LD
  • URLs limpias y coherentes
  • Estructura de headings correcta
  • Alt text en todas las imágenes

Accesibilidad

  • Navegación por teclado funcional
  • Screen reader compatible
  • Contraste de colores adecuado
  • Textos alternativos descriptivos
  • Estructura semántica correcta

Marca y Consistencia

  • Uso correcto del logo y colores
  • Tipografía consistente en todo el sitio
  • Espaciados uniformes
  • Tono de voz coherente
  • Elementos visuales alineados
